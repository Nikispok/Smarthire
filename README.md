SmartHire — Job Application Tracker

A full-stack web application for tracking job applications with AI-powered career advice.

Live Demo: https://smarthire-jod4yalw3-nikispoks-projects.vercel.app

---

Features

- User registration and authentication with JWT tokens
- Add, edit, and delete job applications
- Track application status (Applied / Rejected / Offered)
- AI career advice for each application powered by Groq
- Each user sees only their own applications
- Responsive dark UI

Tech Stack

Backend
- Python + FastAPI
- PostgreSQL + SQLAlchemy
- JWT authentication (PyJWT)
- Password hashing (bcrypt + passlib)
- Groq AI API
- Deployed on Railway

Frontend
- React + Vite
- React Router
- Deployed on Vercel
